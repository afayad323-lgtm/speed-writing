# Typing Speed Test Game

A fun and interactive typing speed game with three difficulty levels: Easy, Normal, and Hard. Players must type the displayed words as fast as possible before the time runs out.

## Features

- Three levels: Easy, Normal, Hard  
- Words to type are different for each level  
- Time-limited typing for each word  
- Score tracking and high score per level using Local Storage  
- Restart game option  
- Responsive and clean UI

## How to Play

1. Select a difficulty level from the dropdown.  
2. Click "Start Playing" or press Enter.  
3. Type the word displayed in the input field before the timer ends.  
4. If typed correctly, the word is removed and your score increases.  
5. Continue until all words are typed or time runs out.  
6. View your high score for the current level.
